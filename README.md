# SQL Problem Solving

This repository contains my SQL solutions for problems from LeetCode, HackerRank, and Programmers.

The purpose of this repository is to document how I interpret problem requirements and choose SQL patterns.

Also, I try to improve query readability and efficiency.



## Repository Purpose

This repository is maintained to:

1. Practice SQL through problem solving
2. Organize solutions by platform and difficulty
3. Review frequently used SQL patterns
4. Understand differences between SQL dialects
5. Build practical SQL skills for data analysis and data science roles
